Alternative Import Methods

Using Draw.io

Import to Draw.io
  1. Add Draw.io diagram to your Confluence page
  2. Click "Arrange" → "Insert" → "Advanced" → "Mermaid"
  3. Paste the Mermaid code
  4. The diagram will be converted to Draw.io format
  5. Save and publish

Using Images

Convert to Images

For PlantUML:

  1. Go to PlantUML Online
  2. Paste the .puml content
  3. Click "Submit"
  4. Right-click and save the image
  5. Upload to Confluence

For Mermaid:

  1. Go to Mermaid Live Editor
  2. Paste the .mmd content
  3. Click "Download PNG" or "Download SVG"
  4. Upload to Confluence
