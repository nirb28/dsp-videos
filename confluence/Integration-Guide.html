Troubleshooting

Services Won't Start

Problem: Docker containers fail to start

Solutions:

  • Check port conflicts: netstat -an | findstr "8000 8080 5000"
  • Verify Docker has enough resources (8GB RAM minimum)
  • Check logs: docker-compose logs
  • Ensure .env file exists and is properly formatted
  • Try clean restart: docker-compose down -v && docker-compose up -d
 JWT Token Validation Fails

Problem: Tokens rejected by services

Solutions:

  • Verify JWT_SECRET_KEY is identical in all services
  • Check token hasn't expired
  • Ensure JWT_ALGORITHM matches (HS256)
  • Review JWT Service logs for generation errors
  • Test token with decode endpoint
 Routing Not Working

Problem: Requests not reaching backend

Solutions:

  • Verify manifest exists in Control Tower
  • Check module name in URL matches manifest
  • Review Front Door logs for routing decisions
  • Ensure backend service is accessible from Front Door
  • Test Control Tower connectivity from Front Door
